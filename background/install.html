Thank you for using the Web Extension Boilerplate

This boilerplate is meant for building Firefox and Chromium extensions. File bugs/issues with this boilerplate here

Debugging about:debugging chrome://extensions
Testing
  • npm run test runs web-ext run with added flags to ensure that the browser launches
  • npm run build runs web-ext build
  • npm run lint runs web-ext lint
  • npm run test-no-tmp runs web-ext run without added flags

By default, in Firefox, host permissions must be granted for a content script to execute on all urls. Otherwise, it will only execute when the popup is opened.
