Thank you for using the Chrome Extension Boilerplate

This boilerplate is designed to help you build extensions for both Firefox and Chromium. If you encounter any bugs or issues, feel free to report them here .

Testing
  • npm run dev runs web-ext run -t chromium to run the extension for Chromium
  • npm run build runs web-ext build to build for Chromium
  • npm run lint runs web-ext lint to lint your repository

    • Firefox Specific:
  • npm run dev:firefox runs web-ext run --config=firefox-config.js to run the extension for Firefox with a config file specific to Firefox
  • npm run build runs web-ext build --config=firefox-config.js to build for Firefox with its specific config file

Note for Manifest Version 3

By default, in Firefox, host permissions must be granted for a content script to execute on all URLs. Otherwise, it will only execute when the popup is opened. This is new for Manifest Version 3.
